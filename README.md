Deskripsi
Aplikasi sederhana untuk menampilkan informasi cuaca berdasarkan nama kota menggunakan API dari OpenWeather.

Fitur
Input nama kota
Menampilkan suhu (°C)
Data real-time dari API

Teknologi
Node.js
Express.js
HTML, CSS, JavaScript

API
Menggunakan OpenWeather API:
https://openweathermap.org/api

Cara Menjalankan
Install dependency:
npm install
Jalankan server:
node app.js
Buka browser:
http://localhost:3000

Endpoint
GET /api/cuaca?kota=nama_kota , nama kotanya hanya ada dua (Jakarta dan Surabya)
